solucion

1. Servidor DNS: Es el sistema que traduce los nombres de dominio (como google.com) en direcciones IP numéricas para que las computadoras se entiendan.b. IP: Es una dirección única que identifica a un dispositivo en internet o en una red local, permitiendo que la información llegue al destino correcto.c. Protocolo: Es un conjunto de reglas y estándares que permiten que los dispositivos se comuniquen entre sí de manera ordenada (ejemplo: HTTP).d. Nombre de dominio: Es el nombre amigable que usamos para identificar un sitio web (ejemplo: youtube.com) en lugar de usar una dirección IP difícil de recordar.e. Directorio y nombre de archivo: Es la ruta específica dentro de un servidor que indica en qué carpeta está guardado un archivo y cómo se llama (ejemplo: /imagenes/foto.jpg).

2. Diseñador UI/UX: El diseñador UI se encarga de la apariencia visual (colores, botones), mientras que el UX se asegura de que la navegación sea fácil y lógica para el usuario.b. Cloud Manager / Ingeniero de infraestructura: Gestiona los servidores y servicios en la nube para asegurar que la web sea rápida, segura y siempre esté disponible.c. Administrador de base de datos: Se encarga de organizar, proteger y mantener el almacenamiento de toda la información que maneja el sitio web.d. Project Manager: Es el encargado de liderar el equipo, organizar las tareas y asegurar que el proyecto se entregue a tiempo y con calidad.

3. Significa que el lenguaje tiene la capacidad de realizar cualquier cálculo lógico o algoritmo que una computadora pueda ejecutar, siempre que tenga suficiente memoria y tiempo. Es capaz de manejar variables, bucles y condiciones de forma universal.

4. No se consideran lenguajes de programación porque no tienen lógica de control (no pueden tomar decisiones, hacer ciclos o cálculos complejos). HTML es un lenguaje de marcado para estructurar contenido y CSS es un lenguaje de diseño para dar estilo visual.

5. Una etiqueta es una instrucción especial rodeada por signos "<" y ">" que le indica al navegador qué tipo de contenido está leyendo (un título, un párrafo, una imagen, etc.) para que sepa cómo mostrarlo en pantalla
